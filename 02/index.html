Розмітка заголовків

Заголовки впорядковують текст, формуючи його візуальну структуру. При використанні заголовків необхідно врахувати їх ієрархію. В заголовки не можна викладати інші елементи.

<h1> - заголовок першого рівня, на сторінці рекомендовано використовувати не більше одного разу.

<h2>, <h3>, <h4>, <h5>, <h6> - заголовки нижчіх рівнів, розташовані за спаданням їх важливості.


Синтаксис:


<h1> Заголовок 1-го рівня </h1>

<h2> Заголовок 2-го рівня </h2>

<h3> Заголовок 3-го рівня </h3>

<h4> Заголовок 4-го рівня </h4>

<h5> Заголовок 5-го рівня </h5>

<h6> Заголовок 6-го рівня </h6>


Результат

Заголовок 1-го рівня

Заголовок 2-го рівня

Заголовок 3-го рівня

Заголовок 4-го рівня

Заголовок 5-го рівня
Заголовок 6-го рівня


Форматування тексту



<strong> — виділяє важливий текст, відображається жирним текстом.

Синтаксис:

Тег визначає <strong> важливість </strong> тексту

Результат:

Тег визначає важливість тексту



<em> — відображає шрифт курсивом, надаючи тексту значення.

Синтаксис:

HTML — <em>головний</em> інструмент веб-розробки

Результат:

HTML — головний інструмент веб-розробки



<small> — зменшує розмір шрифту на одиницю проти звичайним текстом

Синтаксис:

Кішка це зовсім не <small>лев</small>

Результат:

Кішка це зовсім не лев



<ins> — призначений виділення тексту, позначається підкресленням.

Синтаксис:

Доставка коштуе <ins> 60грн </ins> 20грн

Результат:

Доставка коштує 60грн



<del> — призначений виділення тексту, позначається перекресленням слова.

Синтаксис:

Доставка коштує <del>60грн<del> 20грн

Результат:

Доставка коштує 60грн 20грн



<sub> — відображає шрифт у вигляді нижнього індексу

Синтаксис:

Вода — H<sub>2</sub>O

Результат:

Вода — H 2O



<sup> — відображає шрифт у вигляді вверхнього індексу

Синтаксис:

x<sup>2</sup> * y<sup>2</sup> = z <sup>2<sup>

Результат:

x 2 * y 2 = z 2


Введення "комп'ютерного" тексту



<code> — призначений для відображення одного або декількох рядків тексту, що є програмним кодом.

Синтаксис:

<code>function checkParent (src, dest) {
                    while (src != null) {
                        if (src.tagName == dest) return src
                        src = src.parentElement
                    }
                    return null
                }</code>

Результат:

function checkParent (src, dest) {
                    while (src != null) {
                        if (src.tagName == dest) return src
                        src = src.parentElement
                    }
                    return null
                }



<kbd> — використовується для позначення тексту, який набирається на клавіатурі або назви клавіш.

Синтаксис:

Для продовження натисніть <kbd>Enter</kbd>

Результат:

Для продовження натисніть Enter



<samp> — використовується для відображення тексту, що є результатом виведення комп'ютерної програми або скрипта.

Синтаксис:

<samp>Помилка #2314458</samp>

Результат:

Помилка #2314458



<pre> — відображає усі пробіли написані в коді, а не лише один

Синтаксис:

<pre> 2      3      4      5</pre>

Результат:

2    3    4    5


Оформлення цитат та визначень



<dfn> — застосовується для виділення таких термінів за їх першої появи в тексті.

Синтаксис:

<dfn>Уран</dfn> — сьома від Сонця велика планета Сонячної системи

Результат:

Уран — сьома від Сонця велика планета Сонячної системи



<abbr> — казує, що послідовність символів є абревіатурою. За допомогою атрибуту title дається розшифровка скорочення, що дозволяє розуміти абревіатуру тим людям, які з нею не знайомі.

Синтаксис:

<abbr title = "National Aeronautics and Space Administration">NASA<abbr> — агентство уряду США, засноване 1958 року для досліджень у галузі аеронавтики й космічних польотів.

Результат:

NASA — агентство уряду США, засноване 1958 року для досліджень у галузі аеронавтики й космічних польотів.



<q> — використовується для виділення цитат

Синтаксис:

Альберт Эйнштейн: <q>Стремитесь не к успеху, а к ценностям, которые он дает</q>

Результат:

Альберт Эйнштейн Стремитесь не к успеху, а к ценностям, которые он дает



<cite> — представляє назву творчої роботи (книги, статті, поеми, сценарію, фільму, пісні, опери, ігри та ін).

Синтаксис:

Книга <cite>Гаррі Поттер</cite> має 1032 ріцензії

Результат:

Книга Гаррі Поттер має 1032 ріцензії


<blockquote> — призначений для виділення довгих цитат усередині документа.

Синтаксис:

<blockquote>Каждый день используйте свое зрение так, будто завтра вас поразит слепота. Слушайте звуки голосов,
птичье пение так, будто завтра вы оглохнете навсегда. Вдыхайте аромат цветов, наслаждайтесь вкусом пищи
так, будто завтра вы потеряете обоняние и все для вас станет пресным.<blockquote>

Результат:

Каждый день используйте свое зрение так, будто завтра вас поразит слепота. Слушайте звуки голосов,
птичье пение так, будто завтра вы оглохнете навсегда. Вдыхайте аромат цветов, наслаждайтесь вкусом пищи
так, будто завтра вы потеряете обоняние и все для вас станет пресным.


<br> — становлює переклад рядка в тому місці, де він знаходиться.

Синтаксис:

1<br>2<br>3

Результат:

1
2
3

HTML — списки

<li> — визначає окремі пункти списку. Використову разом з тегами <ol>(нумерований список) та <ul> (маркований список)


Нумерований список

<ol> — встановлює, що список буде нумерованим

Синтаксис:

                    <ol>
                        <li>Хліб</li>
                        <li>Сир</li>
                        <li>Масло</li>
                    </ol>

Результат:

  1. Хліб
  2. Сир
  3. Масло


Маркований список

<ul> — встановлює, що список буде маркованим

Синтаксис:

                    <ul>
                        <li>Хліб</li>
                        <li>Сир</li>
                        <li>Масло</li>
                    </ul>

Результат:

  • Хліб
  • Сир
  • Масло


HTML — силки



<a> — є одним з важливих в HTML і призначений для створення посилань. Для цього необхідно повідомити браузер, що є посиланням, а також вказати адресу документа, на який слід зробити посилання. Як атрибут href використовується адреса документа, на який відбувається перехід.

Синтаксис:

<a href="01/task_3.html">Завдання 3. Так ніхно не кохав ..</a>

Результат:

Завдання 3. Так ніхно не кохав ..


HTML — зображення



<img> — призначений для відображення на веб-сторінці зображень у графічному форматі GIF, JPEG, SVG або PNG. Адреса файлу з зображенням задається через атрибут src .

Синтаксис:

<img src = "../01/img/one.png">

Спеціальні символи



&nbsp; — нерозривний пробіл

Синтаксис:

Марія&nbsp;Іванівна

Результат:

Марія Іванівна



&copy; — символ copyright

Синтаксис:

&copy;Україна

Результат:

©Україна



&trade; — знак торговой марки

Синтаксис:

&trade;Україна

Результат:

™Україна



&amp; — символ амперсанта

Синтаксис:

&amp;copy;

Результат:

&copy;



&lt; — знак "меньше"

Синтаксис:

3&lt;5

Результат:

3<5



&gt; — знак "більше"

Синтаксис:

5&gt;3

Результат:

5>3
